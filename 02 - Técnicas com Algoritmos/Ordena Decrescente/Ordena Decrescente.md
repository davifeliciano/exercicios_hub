### Ordena Decrescente ###

Escreva uma função chamada ordenaNumeros(numeros) que recebe um array de números como parâmetro e retorna um array dos números ordenados em ordem decrescente.

Exemplo:

﻿

* numeros = [1, 2, 4, 10, 0] =\> Deve retornar [10, 4, 2, 1, 0]
* numeros = [] =\> Deve retornar []
